# Xio Releases

## 项目简介
`Xio` 是一款以“出牌 + 装备”规则为核心的对战小游戏，包含微信小程序前端与 Python 后端。本项目支持单人对战与联机对战。

## 版本索引
- `1.2.3`
  - 文档：`release/1.2.3/版本说明书.md`
  - 说明：联机大厅 + 房间式联机流程稳定版（含等待弹窗、准备流程、15 秒计时）
- `1.1.1`
  - 文档：`release/1.1.1/版本说明书.md`
  - 说明：战斗页真机适配与交互稳定版
